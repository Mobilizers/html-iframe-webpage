3. Tree-Structure in HTML

In HTML, every element is structured as "tree". An element can have any number of other elements ( within itself) called child elements. To understand this new jargons, watch this video.

4. What is Boxes, grids in HTML documents

A webpage is divided into boxes, each box is divided by the <div> element. each box can hold any data not limited to images, text, videos. Why we use a box structure? it is because a box is the only structure that can use all the space available in a webbrowser and also, the screen displaying the webpage is designed with pixels

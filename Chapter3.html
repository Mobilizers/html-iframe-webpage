3. CSS selectors

A Css selector determines what style is being applied to the document. They are two types of selectors
  1. Id selector: An Id selector can be defined to every Html element individually, it can be defined in seperated CSS document or can be defined with in the respective tag using style attribute within the tag.
  2. Class selector: In short, A class selector can be used to defined "set of defintions" to a "perticulat tag" within a perticular document.

1. How the Web Works

The web is a bunch of computers that communicate with each other. It is composed largely of HTML documents where HTML stands for Hyper Text Markup Language and it is the language the browsers use to interpret web pages. The components of the web include the client (your computer and browser), the Internet, and servers. The client and servers interact through Hypertext Transfer Protocol (HTTP).
      For instance, When a person goes to a web page like www.wikipedia.com on the computer browser, then the browser sends a HTTP Request via Internet to the Server. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and display it on the user's screen. 2. HTML Basics

What is HTML (Hyper Text Markup Language) really made up of? They are 4 main components of HTML,which are as follows:
  1. Basic text content - It defines "what you see", the text seen by the user in unedited form.
  2. Mark up language - It defines "what the content should look like",like headings, bold text,.
  3. References to other pages - It enable a link to other Html documents related to the content.
  4. Reference to images and videos - This enables to insert images and video content into the page.

3. HTML Tags

HTML language is defined by tags, they are structured as below;
opening tag represented as        <tag_name>
"All the text content visible to user"
closing tag represented as      </tag_name>
Html structure with closing and ending tags is called as an "Element".

Element: <tag_name> content </tag_name> HTML elements are used to tell a web browser, "how to display content on a web page".
This video helps in understand the concept about element.

Different types of elements discussed;

  1. <b> tag - This tag makes the text bold or thick. For instance: Hello! this is an experiment text. To make the "experiment text" look bold we use <b > tag to that perticular text Hello! this is an experiment text.
  2. <em> tag- This tag emphasis the text by making it italic Example Text: Hello! "there this is an example text". To make "example" in the sentence emphasized we apply <em> tag it gives the following result:
    Hello! there this is an example text.
  3. <p> tag- Defines a paragraph. The content of the <p> element is created inside a "box".
  4. <br>tag - Creates a line break, and starts a new line.
  5. <div> tag- Creates a "box" container element.
  6. <span>tag- Creates an "inline" container element.
  7. <img> tag- This tag is used to insert images into the document.
  8. <a> tag- This tag is called anchor tag, its used to embed another pages within the current document by creating a link.

5. Inline Vs. block element

Inline elements are those which can be executed anywhere on the document but the output is reflected on one line only.

For reference: <br> break tag, <b> bold tag, <em> empasis tag.

Block elements are those which once created, all the data inserted in it will be within a box.

6. HTML Attributes

In Opening tags, some elements accept additional values called attributes. Attributes provide additional information to the browser.
For example, the anchor tag takes an attribute called href that defines a URL to link to. <a href="url"> link </a> Creates a hyperlink to the href attribute value url

and similarly, to names its link <img src="url" alt="text"> Displays the image hosted at the src attribute value "url" (this can also be a path to a local file on your computer). If the image cannot be displayed, then the alt attribute value "text" is shown.
